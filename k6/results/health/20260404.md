# 06-health 부하테스트 — 2026-04-04

환경: K8s (Docker Desktop) + Docker Compose 인프라
VU: 5 | Duration: 30s

## 1차 결과: ⏭️ SKIP

K8s 환경에서 개별 서비스 포트(8761, 8081-8085)가 LoadBalancer로 노출되지 않아 전부 connection refused.
Gateway(8080)와 pm-web(3000)만 외부 접근 가능.

## 수정 내역

1. **Gateway 라우팅 추가**: `/api/health/{service}` → `SetPath=/health`로 각 서비스 헬스체크 프록시
2. **JWT 인증 우회**: `/api/health` 를 OPEN_ENDPOINTS에 추가
3. **k6 테스트 리팩토링**: 직접 서비스 포트 접근 → Gateway 경유 방식으로 변경
4. **Gateway 헬스체크 제거**: actuator 미노출로 gateway 자체 헬스체크 제외 (다른 서비스 성공 = gateway 정상)
